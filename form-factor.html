:user-valid / :user-invalid

These inputs use :user-valid/:user-invalid if supported; otherwise they fall back to :valid/:invalid.

:user-* only applies after user interaction—preventing “everything red on page load.” Fallback keeps broad support.

@supports selector(:user-invalid) {
  .form input:user-invalid { border-color: var(--invalid); }
  .form input:user-valid { border-color: var(--valid); }
}
